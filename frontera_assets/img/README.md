# Frontera - Images

## How to Load Images

- As a CDN path for [`PORTAL_LOGO` and `PORTAL_FAVICON` in Frontera's CMS settings](https://github.com/TACC/Core-Portal-Deployments/blob/d6af7b2/frontera-portal/camino/cms.settings_custom.py#L32-L45).
- Import from within a stylesheet in this repository via a relative path to this directory.
- Load via a Django CMS plugin on a CMS page.
